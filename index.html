Pack de Iconos Web Modernos

Colección de iconos SVG escalables y personalizables para tus proyectos web

🚀 ¿Cómo usar estos iconos?

  • 1. Instala o descarga: Haz clic en el botón para descargar todos los iconos en formato SVG.
  • 2. Inserta en tu HTML: Puedes usar el archivo directamente o integrarlo mediante un CDN.
    <script src="https://cdn.jsdelivr.net/gh/rodrigoespigares/rodricon@main/src/js/icons.js"/>
    <script src="https://cdn.jsdelivr.net/gh/rodrigoespigares/rodricon@main/src/js/index.js"/>
  • 3. Personaliza: Cambia colores, tamaños y efectos fácilmente con CSS.
    .icon { width: 24px; fill: #7c3aed; }
    O usando clases de bootstrap.
  • 4. Clic para copiar: Haz clic en cualquier icono para copiar las clases del icono en tu portapapeles.
